<h1>Network Remote Control</h1>

<h2>Description</h2>
Automation script that allows cyber units to run script from their devices but would be executed by the remote server anonymously.
<br />


<h2>Languages Used</h2>

- <b>Bash</b>

<h2>Environments Used </h2>

- <b>Linux</b>
